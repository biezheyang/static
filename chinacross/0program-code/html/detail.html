2011年度最大型的越野汽车“无人区穿越”活动：北京——甘肃省敦煌，2500公里左右为行驶路段；敦煌——新疆罗布泊湖心——哈密，1250公里左右为罗布泊无人区穿越路段；哈密——内蒙古额济纳，800公里左右为黑戈壁无人区穿越路段；额济纳——北京1700公里左右为行驶线路段。途经北京、河北、山西、陕西、宁夏、甘肃、新疆、内蒙古等八个省市自治区。活动将历时十五天左右。参与活动的人员众多，有68人，其中女队员为12人；年龄最大的为57岁，最小的为20岁；这其中有父女、夫妻和亲兄弟等多对搭档。他们均是各行业的优秀人士，其中还有新闻工作者、国家级专业裁判和赛车手等。是2011年度我们已知的国内最大型的越野汽车穿越和挑战活动。

在组织本次活动之初，我们就与中国盲文图书馆协商组织一次关爱边疆地区盲人工作的公益活动，最后，我们选定了哈密市，这个有着众多民族，并且能和谐共进的美丽城市。当经过近十天，经过八千里路云和月的艰难行程之后，我们把中国盲文图书馆提供的爱心传递的物资——盲文书籍交到哈密地区残疾人联合会、兵团十三师残联和哈密地区盲人协会领导及盲人朋友手里时，我们共同感受到了那种血浓于水的爱之力量，那种中华民族大家庭的难以言表的亲情。在现场，大家都流下了激动的眼泪，这让我想起诗人艾青曾经说过的一句话：为什么我的眼里常含泪水？因为我对这土地爱得深沉。
